# Jester Wyrm Court

Explorable 2.5D isometric fantasy map prototype with a dark purple jester and dragon theme.

## Run

Open `index.html` directly, or serve the folder locally:

```sh
python3 -m http.server 8765
```

Then visit:

```text
http://127.0.0.1:8765/
```

## Controls

- Drag to pan the map.
- Mouse wheel to zoom.
- Click a landmark to inspect it.
- Click the mini-map to jump to the nearest landmark.
- Press `N` for the next landmark.
- Press `R` to reset the view.
