# Drako Lair Isometric Map

Explorable 2.5D isometric Drako Lair macro-map. The old court zone now works as one local district inside a larger world with aquatic, volcanic, mineral, forest, underground, creature-trace, and gothic architectural sectors.

## Run

Open `index.html` directly, or serve the folder locally:

```sh
python3 -m http.server 8765
```

Then visit:

```text
http://127.0.0.1:8765/
```

## Controls

- Drag to pan the map.
- Mouse wheel to zoom.
- Click a landmark to inspect it.
- Click the mini-map to jump to the nearest landmark.
- Press `N` for the next landmark.
- Press `R` to reset the view.
- Press `G` or `V` to switch between atlas view and game camera view.
